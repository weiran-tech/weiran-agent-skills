# Prompt Optimizer

启动提示词优化流程。将模糊的想法转化为精确有效的 AI 提示词。

## 使用方式

提供你的原始提示词或需求描述：

## 支持的平台

- **Claude** - XML 标签结构
- **ChatGPT** - Markdown 结构
- **DeepSeek** - CoT 友好格式
- **豆包** - 简洁中文风格
- **智谱 GLM** - 结构化中文
- **Gemini** - 清晰指令格式

## 优化方法论（4D）

1. **Define** - 明确意图和目标
2. **Decompose** - 拆解复杂任务
3. **Demonstrate** - 添加示例
4. **Debug** - 迭代优化
