# PRD Studio

启动 PRD 全自动工作室。自动完成「写 PRD → 审查 → 修改 → 再审」的完整循环，无需人工干预。

## 使用方式

描述你想要的功能或需求：

$ARGUMENTS

## 工作原理

1. **Writer 阶段**：独立 subagent 撰写 PRD 初稿
2. **Reviewer 阶段**：独立 subagent 审查 PRD
3. **判断**：P0=0 且 P1<2 → 准出；否则继续
4. **Fixer 阶段**：独立 subagent 修复问题
5. **循环**：最多 3 轮，直到准出或达到上限

## 核心优势

- **隔离执行**：每个阶段在独立上下文中执行，避免污染
- **全自动流转**：无需人工干预
- **质量保证**：严格的准出标准

## 输出

- 准出通过 → PRD 文档 + 准出证书
- 有遗留问题 → PRD 文档 + 遗留问题报告
